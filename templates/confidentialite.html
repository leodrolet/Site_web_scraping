Politique de confidentialité

Dernière mise à jour : 2025

Données que nous collectons

  • Votre adresse courriel et, si vous le souhaitez, votre nom.
  • Votre mot de passe, stocké uniquement sous forme de hash (bcrypt) — jamais en clair.
  • Vos clés API (Hunter.io, Apollo.io, SerpAPI), stockées chiffrées (Fernet).
  • L'historique de vos recherches (nom d'entreprise, département, région, nombre de contacts trouvés).

Utilisation des données

Vos clés API servent exclusivement à exécuter vos propres recherches via Hunter.io, Apollo.io et Google (SerpAPI). Nous ne partageons ni ne vendons vos données à des tiers.

Vos droits

Vous pouvez à tout moment modifier ou supprimer vos clés API depuis la page Paramètres. Pour supprimer votre compte et l'ensemble de vos données, contactez-nous à contact@prospectb2b.app.

Sécurité

Les mots de passe sont hachés avec bcrypt, les clés API chiffrées avec Fernet, et les sessions protégées par des cookies signés expirant après 7 jours. Les formulaires sont protégés contre la falsification de requête (CSRF).
